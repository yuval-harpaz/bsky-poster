# ![Icon](public/icon.png) bsky-poster
Post rich text and links to bluesky.
The web app is available at [bsky-poster](https://yuval-harpaz.github.io/bsky-poster/).<br>
I created this app so I could insert links with ctrl+k, without showing the full url.<br>You can also use [markdown]\(links\) and \*\***style**\*\*  .
Developed with [Gemini 3](https://gemini.google.com/app) Pro on [Antigravity](https://antigravity.google/).
